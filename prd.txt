# 사주 운세 전문 앱 PRD - "운명나침반"

## 1. 프로젝트 개요

### 1.1 프로젝트 목표
전통 명리학에 기반한 정확하고 신뢰성 있는 사주 운세 앱을 개발하여, 일반 사용자부터 전문 상담사까지 모두가 활용할 수 있는 포괄적인 플랫폼 구축

### 1.2 시장 현황
- **글로벌 점성술 앱 시장**: 2024년 40억 달러 → 2030년 90억 달러 예상 (CAGR 20%)
- **한국 시장 특성**: 전통 명리학에 대한 높은 관심, 디지털 전환 가속화
- **경쟁 현황**: 기본적인 만세력 제공 앱들이 대부분, 전문적 분석 도구 부족

### 1.3 핵심 차별화 포인트
1. **정확성**: NASA 천문 데이터 기반 2200년치 만세력 데이터
2. **전문성**: 120개 이상 신살, 완전한 십이운성 시스템 구현
3. **AI 융합**: 전통 명리학 + 현대 AI 기술의 결합
4. **사용성**: 초보자부터 전문가까지 만족하는 다층적 UX
5. **품질**: 검증된 전문가들의 감수를 거친 해석 시스템

## 2. 타겟 사용자 분석

### 2.1 Primary Target (60%)
**일반 사용자 - 운세 관심층**
- 연령: 30-50대 여성
- 특성: 정기적으로 운세를 확인하는 습관, 모바일 앱 활용도 높음
- 니즈: 정확하고 이해하기 쉬운 운세 정보, 일상생활 가이드

### 2.2 Secondary Target (25%)
**젊은 층 - 신규 유입층**
- 연령: 20-30대 남녀
- 특성: 새로운 경험에 열린 마음, SNS 활용도 높음
- 니즈: 재미있고 트렌디한 운세 콘텐츠, 친구들과 공유 가능한 기능

### 2.3 Tertiary Target (15%)
**전문가 - B2B 시장**
- 대상: 역술인, 상담사, 철학관 운영자
- 특성: 전문적 도구 필요, 정확성과 신뢰성 중시
- 니즈: 완전한 분석 도구, 고객 관리 기능, 상담 지원 시스템

## 3. 핵심 기능 명세

### 3.1 기본 기능 (무료)
**만세력 및 사주 추출**
- 생년월일시 입력 → 정확한 사주팔자 자동 생성
- 지역별 시차 보정 (전 세계 주요 도시 지원)
- 음력/양력 자동 변환 시스템
- 절입시간 정확 적용

**🌟 혁신 기능: 운세 기반 스마트 다이어리**
- **상단 운세 정보**: 오늘의 운세, 길일/흉일, 주의사항, 추천 활동
- **중앙 다이어리 공간**: 자유로운 개인 기록 및 일정 관리
- **하단 인사이트**: 오행 기반 기분 체크, 하루 요약
- **운세 토글**: 필요에 따라 운세 정보 on/off 가능
- **일/주/월/년 뷰**: 글로벌 시간 관리 도구 수준의 캘린더 시스템

**기본 사주 분석**
- 오행 분석 및 시각화
- 기본 십성(십신) 표시 및 설명
- 간단한 성격 분석
- 오늘의 운세 (일진 기반)

### 3.2 프리미엄 기능 (유료)
**고급 명리학 분석**
- 120개 이상 신살 완전 구현
- 십이운성 상세 분석
- 용신/기신 분석 시스템
- 격국 분석 및 해석

**운세 분석 시스템**
- 대운(10년 단위) 상세 분석
- 세운(1년 단위) 길흉 판단
- 월운/일운 세밀한 예측
- 평생 주기 운세 리포트

**🌟 고급 다이어리 기능**
- **운세 기반 일정 관리**: 중요한 일을 길한 날에 배치하는 AI 추천
- **스마트 알림**: 개인별 주의해야 할 날짜 및 활동 자동 알림
- **장기 계획 도구**: 대운/세운 기반 연간 목표 설정 및 추적
- **템플릿 시스템**: 운세 기반 맞춤형 다이어리 템플릿 제공
- **데이터 분석**: 운세와 실제 일상의 상관관계 인사이트
- **클라우드 동기화**: 모든 디바이스에서 실시간 접근
- **오프라인 모드**: 인터넷 없이도 다이어리 작성 가능

**AI 기반 개인화**
- GPT 기반 개인별 맞춤 해석
- 자연어로 된 상세한 운세 설명
- 생활 가이드 및 조언 제공
- 중요한 시기 알림 기능

### 3.3 전문가 기능 (B2B)
**프로 분석 도구**
- 고급 합충형파 분석
- 복잡한 신살 조합 해석
- 다양한 유파별 해석법 지원
- 상세한 육친 분석

**🌟 전문가용 고객 관리 다이어리 시스템**
- **고객별 운세 달력**: 각 고객의 특별한 날짜 자동 표시 및 알림
- **상담 최적 시기 추천**: 고객과 상담사 궁합을 고려한 예약 시간 AI 추천
- **고객 히스토리 관리**: 과거 상담 내용과 운세 변화 추적
- **자동 알림 시스템**: 고객의 중요한 길일/흉일 사전 알림
- **상담 준비 도구**: 고객별 맞춤 상담 자료 자동 생성
- **CRM 통합**: 고객 연락처, 상담 이력, 결제 정보 통합 관리
- **리포트 생성**: 고객별 맞춤 운세 리포트 자동 작성

**상담 지원 시스템**
- 고객 사주 데이터 관리
- 상담 기록 저장 및 관리
- 리포트 자동 생성 기능
- 예약 및 결제 시스템

### 3.4 커뮤니티 기능
**소셜 기능**
- 궁합 분석 (친구, 연인, 가족)
- 운세 공유 및 SNS 연동
- 명리학 학습 커뮤니티
- 전문가 Q&A 게시판

## 5. UI/UX 설계 방향

### 5.1 디자인 철학
**전통과 현대의 조화**
- 한국 전통색을 활용한 프리미엄 색상 팔레트
- 금색, 적색, 검정을 베이스로 한 고급스러운 톤
- 한글 타이포그래피 중심의 가독성 있는 디자인
- 다크모드 지원으로 신비로운 분위기 연출

### 5.2 사용자 경험 설계
**단계별 정보 제공**
1. 간단한 온보딩으로 기본 정보 수집
2. 즉시 볼 수 있는 핵심 사주 정보 제시
3. 점진적으로 더 자세한 분석 제공
4. 사용자 레벨에 따른 정보 깊이 조절

**직관적인 네비게이션**
- 탭 기반의 명확한 정보 구조
- 한 눈에 파악 가능한 아이콘 시스템
- 터치 제스처를 활용한 편리한 조작
- 음성 가이드 지원 (접근성 고려)

### 5.3 핵심 화면 구성
**메인 대시보드**
- 오늘의 운세 요약
- 현재 대운/세운 상태 표시
- 중요한 길일/주의일 알림
- 개인화된 조언 카드

**🌟 스마트 다이어리 화면**
- **상단 운세 섹션**: 오늘의 운세, 길흉, 주의사항, 추천 활동
- **메인 다이어리 공간**: 
  - 자유 텍스트 입력 (마크다운 지원)
  - 사진/이미지 첨부
  - 음성 메모 기능
  - 태그 및 카테고리 시스템
- **하단 인사이트**: 
  - 오행 기반 기분/컨디션 체크
  - 하루 요약 및 성찰 프롬프트
  - 내일의 운세 미리보기
- **캘린더 뷰**: 
  - 월간/주간/일간 전환
  - 운세 정보 색상 코딩
  - 일정과 운세의 시각적 매칭
- **운세 토글**: 언제든 운세 정보 표시/숨김 가능

**전문가 고객 관리 화면**
- **고객 대시보드**: 모든 고객의 오늘의 상태 한눈에 보기
- **고객별 캘린더**: 개별 고객의 연간 운세 달력
- **알림 센터**: 고객별 중요 일정 및 상담 시기 추천
- **상담 스케줄러**: AI 추천 시간대와 고객 선호도 매칭

**사주 분석 화면**
- 사주팔자 시각화
- 오행 밸런스 차트
- 십성 관계도
- 신살 및 십이운성 표시

## 6. 비즈니스 모델

### 6.1 수익 구조
**Freemium 모델**
- 기본 사주 및 간단한 운세: 무료
- 기본 다이어리 기능: 무료 (월 30회 제한)
- 고급 분석 및 AI 해석: 유료 구독
- 무제한 다이어리 + 고급 템플릿: 유료 구독
- 1:1 전문가 상담: 별도 과금

**구독 요금제**
- 베이직: 무료 (광고 포함, 기본 기능, 다이어리 월 30회)
- 프리미엄: 월 12,900원 (고급 분석, 무제한 다이어리, 스마트 알림, 광고 제거)
- 프로: 월 39,900원 (전문가 도구, CRM 기능, 무제한 고객 관리)
- 연간 구독 시 25% 할인 혜택

**추가 수익원**
- 프리미엄 다이어리 템플릿: 개당 2,900원
- 맞춤형 연간 운세 리포트: 개당 19,900원
- 가족/팀 다이어리 공유: 월 4,900원
- **전문가 상담 플랫폼**
  - 1:1 상담: 30,000~100,000원 (상담사별 차등)
  - 플랫폼 수수료: 30%
  - 상담사 검증 및 품질 관리 시스템

### 6.2 예상 수익
**목표 지표 (출시 1년차)**
- MAU: 150,000명 (다이어리 기능으로 retention 300% 향상)
- 유료 전환율: 6% (다이어리 습관성으로 전환율 상승)
- 월 구독 수익: 약 1억 2,000만원
- 다이어리 템플릿/리포트 수익: 약 2,000만원
- 상담 플랫폼 수익: 약 1,500만원
- **총 예상 월 매출: 1억 5,500만원**

**시장 확장 효과**
- 기존 운세 앱 시장 + 다이어리/플래너 앱 시장 동시 타겟
- 글로벌 다이어리 앱 시장: 약 10억 달러
- 일 평균 사용 빈도: 기존 월 1-2회 → 일 1-2회 (20배 증가)
- Lifetime Value 400% 향상 예상

**소프트 론칭**

**그랜드 론칭**
- 유명 역술인과의 콜라보 이벤트

### 7.2 마케팅 채널
**디지털 마케팅**
- Google Ads / Facebook Ads (타겟 광고)
- 유튜브 / 틱톡 (운세 콘텐츠 마케팅)

- 매일 업데이트되는 운세 콘텐츠

- 명리학 교육 영상 시리즈
- 시즌별 특별 운세 리포트

### 7.3 파트너십 전략
**전문가 네트워크**
- 검증된 역술인 100명 이상 확보 (다이어리 CRM 기능으로 유치)
- 전문가 교육 프로그램 운영
- 품질 관리 및 평가 시스템

**🌟 다이어리 생태계 파트너십**
- **문구/스테이셔너리 브랜드**: 오프라인 다이어리와 연동 서비스
- **웰니스 업체**: 명상, 요가 센터와의 운세 기반 프로그램 제휴
- **기업 HR**: 직원 웰빙 프로그램으로 B2B2C 진출
- **교육 기관**: 자기계발, 라이프 코칭 업체와 제휴

**B2B 제휴**
- 기존 철학관 / 역술원과 제휴 강화
- 웨딩업체와의 궁합 + 결혼 준비 다이어리 서비스
- 기업 상담 서비스 제공
- **새로운 B2B 영역**: 
  - 펜션/호텔: 투숙객 맞춤 여행 일정 추천
  - 이벤트 기획사: 최적의 행사 날짜 컨설팅
  - 부동산: 입주 길일 및 인테리어 타이밍 컨설팅

5. **검증된 전문가 네트워크**: 품질과 신뢰성 보장
6. **습관형성 도구**: 일상적 사용을 통한 높은 사용자 retention

### 11.2 다이어리 기능의 게임체인저 효과
**시장 확장**
- 기존 운세 앱 시장 (40억$) + 다이어리 앱 시장 (10억$) 동시 타겟
- 크로스오버 신시장 창출로 경쟁자 없는 블루오션 확보

**사용 패턴 혁신**  
- 기존: 월 1-2회 → 새로운: 일 1-2회 (20배 사용 빈도 증가)
- 수동적 정보 소비 → 능동적 일상 계획 도구로 전환
- 개인 도구 → 가족/팀 협업 도구로 확장 가능

**비즈니스 임팩트**
- LTV (Lifetime Value) 400% 증가
- 유료 전환율 50% 향상 (4% → 6%)
- 월간 매출 300% 증가 (5,000만원 → 1억 5,500만원)
