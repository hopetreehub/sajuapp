💡 왜 Root Directory를 packages/web로 설정하나요?

   packages/web/
   ├── api/              → /api로 자동 매핑
   ├── src/              → React 앱
   ├── dist/             → 빌드 출력
   ├── vercel.json       → Vercel 설정
   └── package.json      → 의존성

   Root를 packages/web로 설정하면:
   - Vercel이 이 폴더를 프로젝트 루트로 인식
   - api/ 폴더가 Serverless Functions로 자동 변환
   - vercel.json 설정 자동 적용
   - 빌드 명령 단순화
